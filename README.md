# Real Estate Investment Platform

Using data to make it easier for a normal Australian to make smarter property investing decisions.

## About This Project

This MVP helps first-home buyers and investors identify the right suburbs in Sydney to invest in, using data-driven insights and interactive maps.

## Tech Stack

This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`.

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
